Происхождение имени Бахар: персидское имя.

Возможные положительные качества числа: совершенство, идеализм, любовь, сострадание, впечатлительность, благотворительность, милосердие, беспристрастность, талант художника, писателя, актера, религиозность, удачливость, "притягивание" денег, способность жить по Божественным Стандартам.
